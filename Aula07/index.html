Introdução ao DOM

Objetivo : Compreender o que é o DOM e como o JavaScript interage com elementos HTML.

Pesquisa

1. O que é DOM? Explique com suas palavras e a relação com HTML.

O DOM (Document Object Model) é uma representação estruturada de uma página web que o navegador cria a partir do código HTML, organizando todos os elementos em forma de uma árvore de objetos que podem ser acessados e manipulados; enquanto o HTML define a estrutura inicial da página com suas tags e conteúdos, o DOM transforma essa estrutura em objetos que permitem interação dinâmica, ou seja, por meio dele linguagens como JavaScript conseguem modificar textos, alterar estilos, adicionar ou remover elementos e responder a eventos, fazendo com que a página deixe de ser estática e se torne interativa.


2. O que é um nó (node)? Explique elementos, atributos e textos.

Um nó (node) é cada parte individual que compõe o DOM, ou seja, cada item da estrutura da página é considerado um nó; os elementos são nós que representam as partes da estrutura da página, como títulos, parágrafos e divisões, formando a base do conteúdo; os atributos são informações adicionais desses elementos, como identificadores ou classes, que servem para configurar ou diferenciar cada parte; e os nós de texto são o conteúdo escrito dentro desses elementos, ou seja, aquilo que aparece para o usuário na página, sendo também tratados como nós dentro da árvore do DOM.


3. Explique os métodos: getElementById(), querySelector(), querySelectorAll().

Os métodos getElementById(), querySelector() e querySelectorAll() são usados para selecionar elementos do DOM; o getElementById() serve para buscar um único elemento pelo seu identificador (id), sendo direto e rápido, retornando apenas um elemento; o querySelector() é mais flexível, pois permite selecionar o primeiro elemento que corresponda a um seletor CSS, como classe, id ou tipo de elemento; já o querySelectorAll() retorna todos os elementos que correspondem a um seletor CSS, trazendo uma lista desses elementos, que pode ser percorrida para manipular vários itens ao mesmo tempo.


4. Diferença entre innerHTML e innerText.

A diferença entre innerHTML e innerText está na forma como manipulam o conteúdo de um elemento no DOM; o innerHTML permite acessar ou alterar o conteúdo incluindo a estrutura interna, ou seja, ele interpreta e lê HTML, podendo adicionar ou modificar elementos dentro de outro elemento; já o innerText trabalha apenas com o texto visível para o usuário, ignorando qualquer marcação HTML e considerando apenas o conteúdo textual, sem interpretar tags ou estrutura.

5. O que são eventos? Exemplos: onclick, onchange, onmouseover.

Eventos são ações ou ocorrências que acontecem na página e que podem ser detectadas pelo JavaScript para executar alguma função, geralmente causadas pela interação do usuário ou pelo próprio navegador; por exemplo, onclick acontece quando o usuário clica em um elemento, onchange ocorre quando o valor de um campo é alterado, como em um input ou select, e onmouseover é acionado quando o usuário passa o mouse sobre um elemento, permitindo criar interações dinâmicas na página.
